# MoLBreath-Net

This repository contains the official PyTorch implementation of the paper: "MolBreath-Net: Enhancing Breathomic Lung Disease Diagnosis via Chemical-Aware Deep Learning".

## 🚀 Code Availability
The complete source code, feature engineering pipelines, and model configuration files will be made publicly available in this repository immediately upon the publication of the paper. 

## 📊 Datasets
To ensure full reproducibility, the public clinical breathomics datasets analyzed during this study can be accessed via their respective original repositories:

* **EBC_CF Dataset** (Zang et al., 2020): [MetaboLights MTBLS1786 / doi:10.21228/M8409J](http://dx.doi.org/10.21228/M8409J)
* **EBC_LC Dataset** (Wang et al., 2025): [MetaboLights MTBLS3322 / doi:10.21228/M8JZ41](http://dx.doi.org/10.21228/M8JZ41)
* **CBD Dataset** (Kuo et al., 2024): [Figshare / doi:10.6084/m9.figshare.23522490.v6](https://doi.org/10.6084/m9.figshare.23522490.v6)
